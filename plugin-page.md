# File Template — Plugin marketplace page content

Short description

快速从自定义模板创建文件，提高项目初始化和组件开发效率。

Detailed description

File Template 是一个轻量级的 JetBrains IDE 插件，提供可配置的文件模板管理和一键创建文件功能。主要特性：

- 可在 Settings -> Other Settings -> File Template 中管理模板（名称、文件后缀、类型、内容、是否启用）。
- 模板支持多行内容并保留原有格式（换行/空格）。
- 导入/导出模板（JSON 格式，导入为追加模式）。
- 在 Project 视图右键 -> New 菜单中快速创建文件，创建后自动打开文件。
- 在创建文件对话框和设置页内显示模板序号（从 1 开始），便于识别顺序。

Usage

1. 打开 Settings -> Other Settings -> File Template，点击“新增”添加模板，填写名称、后缀、类型和内容。
2. 在项目视图中右键选择目录，选择 New -> File Template，选择模板并输入文件名，点击 OK 即可创建并打开文件。
