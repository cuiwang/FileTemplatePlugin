# File Template 插件（Marketplace 页面草稿）

简短描述：
File Template 是一款帮助开发者快速基于模版创建文件的 IDEA 插件，支持模版管理、导入/导出以及在 Project 视图中快捷新建文件。

功能亮点：
- 在 Project 视图右键 -> New 中添加“File Template”项，一键创建。 
- Settings -> Other Settings -> File Template 中集中管理模版，支持多行内容、后缀配置与启用开关。
- 支持 JSON 导入/导出模版，便于备份与团队共享。

安装与使用：
1. 安装插件后，打开 Settings -> Other Settings -> File Template 添加模版。
2. 在 Project 视图中右键选中目录 -> New -> File Template -> 填写文件名并选择模版 -> Create。
